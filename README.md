# Proyecto-Aerolinea
Sistema de datos end-to-end para aviación comercial: modelo relacional normalizado (18 entidades, 3FN) + pipeline ETL en Python/SQL Server + 25 queries analíticas avanzadas + dashboard en Streamlit + modelo predictivo de emisiones (CatBoost, R²=0.98) + asistente NL-to-SQL con LLM fine-tuneado localmente (Qwen2.5-Coder + QLoRA).
